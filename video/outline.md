# Enikk 介绍视频大纲（B站科技幽默风）

## 视频信息
- 时长：2-3 分钟
- 风格：科技幽默、接地气、有梗、节奏快
- 目标受众：B站用户、技术宅、摸鱼达人
- 特点：适合弹幕互动、有槽点、有笑点

---

## 大纲结构

### 1. 开场直球 (0:00 - 0:15)
**目标**：快速抓住注意力，不废话

**画面**：黑屏 → 大字闪现

**文案/旁白**：
- "Agent 这么多，为啥还要搞一个？"
- "用 GPT-4？Claude？太贵了吧？"
- "今天介绍一个——不挑模型、能看屏幕、自己进化的 GUI Agent"
- 快速闪过 Enikk Logo

**弹幕预判**：
- "来了来了"
- "直接开整"

### 2. 灵魂拷问 (0:15 - 0:50)
**目标**：正面回答核心问题，建立差异化

#### 拷问 1："Agent 已经满地走了，为啥还要搞一个？"
**文案/旁白**：
- "我知道你们在想啥——'Agent 还少吗？又来一个？'"
- "但大部分 Agent 都是——"
- 快速展示：纯文字对话界面、只能调 API
- "它们很聪明，但它们**看不见**屏幕啊！"
- "Enikk 不一样——它能**看到**你的屏幕，**操作**你的鼠标键盘"
- "比如——" **（高能预警）**
- 快速闪过：Enikk 操作 Codex 写代码的画面
- "没错，AI 操作 AI，套娃了属于是"
- "这才是 GUI Agent，不是 Chat Agent"

**画面重点**：
- 对比：普通 Agent（黑框框）vs Enikk（看到屏幕、操作界面）
- **关键镜头**：Enikk 自动点击 Codex、输入指令、看着它写代码
- "看得见 + 能操作 = GUI Agent"

**弹幕预判**：
- "卧槽，套娃了"
- "AI 自己写代码？"
- "这是 AI 在给自己写代码吧"

#### 拷问 2："用什么模型？GPT-4？"
**文案/旁白**：
- "还有人说——'这得用啥模型啊？GPT-4 吗？用不起'"
- "兄弟，Enikk **不挑模型**"
- "GPT、Claude、Qwen、Llama、本地的 Ollama——都行"
- "甚至！不能识别图片的模型也能用"
- "因为 Enikk 自己就能看图！YOLO + OCR 帮你识别屏幕"
- "模型只需要做决策，不需要看图片"
- "便宜模型照用，本地小模型也能跑——这才是平民自动化"

**画面重点**：
- 模型 Logo 大乱斗
- "来者不拒" 表情包
- 架构图：Enikk 看图 → 文字描述 → LLM 决策

**弹幕预判**：
- "良心！"
- "本地小模型狂喜"
- "终于不用为 GPT-4 破产了"

### 3. 核心功能演示（高能预警）(0:50 - 1:50)
**目标**：用实际操作说话，展示能力

#### 3.1 "它居然能看见" (0:50 - 1:10)

#### 3.1 "它居然能看见" (0:45 - 1:10)
**文案/旁白**：
- "首先，Enikk 有一双慧眼"
- 演示 YOLO + OCR 识别
- "看到没？它不仅能看见按钮在哪，还能认出上面写的是啥"
- "这识别能力，比你找对象的眼光强多了（逃）"

**画面重点**：
- 高亮识别框
- 快速标注文字
- "识别准确率：99%（大概）"

#### 3.2 "它真的会干活" (1:10 - 1:35)
**文案/旁白**：
- "光看见不行，还得会干活"
- 演示自动操作流程：
  - 点击 → 输入 → 等待 → 下一步
- "看好了，全程无尿点"
- "这手速，单身 20 年都练不出来（doge）"

**演示场景**（选一个有趣的）：
1. **基础场景**：自动登录某网站、批量填写表单
2. **高能场景**：**Enikk 操作 Codex 写代码**
   - Enikk 打开 Codex
   - 输入指令："写一个 Python 函数计算斐波那契数列"
   - Codex 开始写代码
   - Enikk 看着它写，偶尔调整
   - "看，AI 在教 AI 写代码，这剧情够科幻吧"
3. **趣味场景**：自动抢票/签到（懂的都懂）

**弹幕预判**：
- "这手速我酸了"
- "AI 自己写代码，离谱"
- "这是在套娃吗？"
- "比我写代码还快（真实）"

#### 3.3 "它啥都能干" (1:35 - 1:50)
- "最骚的是，它不挑食"
- 快速切换演示：
  - 游戏里自动操作
  - 音乐软件自动切歌
  - 浏览器自动爬数据
- "一个打工人，干三份活，还不要求加班费"
- "资本家看了都流泪"

### 4. 技术揭秘（装X时间）(1:50 - 2:10)
**目标**：秀技术肌肉，但保持有趣

**文案/旁白**：
- "吹了半天，来看看它是怎么做到的"
- 架构图快速闪过（配"高端操作"特效）
- "YOLO 看东西，OCR 读文字，AI 想对策"
- "说白了就是：眼睛 + 脑子 + 手"
- "但是！它会自己学习进化"
- "用得越多，越聪明——这不得给它交学费？（笑）"

**画面**：
- 技术栈图标快速闪过
- 简化版架构图
- "自我进化" 动画效果

### 5. 快速演示（有手就行）(2:10 - 2:30)
**目标**：展示易用性，让观众觉得"我也能玩"

**文案/旁白**：
- "说了这么多，怎么用呢？"
- 演示完整流程：
  1. 启动 Enikk → "打开就行"
  2. 创建会话 → "新建任务"
  3. 输入指令 → "告诉它你要干啥"
  4. 观看执行 → "然后就可以摸鱼了"
- "就这么简单，有手就行"
- "不会？那你可能真的需要 Enikk 来教你了（套娃）"

**画面重点**：
- Web UI 操作
- 实际输入指令
- 等待执行（可以加点有趣的等待画面）

### 6. 结尾（三连警告）(2:30 - 2:50)
**目标**：总结 + 要三连 + 彩蛋

**文案/旁白**：
- "好了，这就是 Enikk"
- "一个让你的鼠标可以退休的 AI 助手"
- "开源免费，GitHub 上就能下载"
- "觉得有用的话——"
- "一键三连，让更多人摆脱人肉脚本的命运！"
- "我是 XXX，我们下期见！"

**画面**：
- 功能快速回顾（3-5 秒快剪）
- GitHub 地址展示
- 三连动画
- 关注按钮

**弹幕预判**：
- "已三连"
- "这就去下载"
- "下次一定（不是）"

### 7. 片尾彩蛋 (2:50 - 3:00)
**可选内容**：
- **Meta 彩蛋**："其实这个视频就是 Enikk 操作 Codex 录制的"
- 展示 Enikk 操作 Codex 写代码的快进画面

- "别看了，快去下载！"

## B站特色元素

### 梗和槽点
- "赛博打工人"
- "人肉脚本"
- "单身手速"
- "有手就行"
- "资本家流泪"
- "下次一定"

### 互动点
- 预留弹幕吐槽空间
- "你觉得它能识别出这个吗？"
- "猜猜它下一步要干啥？"

### 节奏控制
- 每 10-15 秒一个小高潮
- 快速转场，不拖沓
- 重要信息重复强调

### 音乐/音效
- 开头：紧张/悬疑 → 转轻松
- 演示部分：节奏感强的电子乐
- 结尾：温暖/励志
- 音效：点击声、完成提示音、转场音效
